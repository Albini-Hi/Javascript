Repetições
"while" (condição) while em português significa ENQUANTO.
"do" significa FAÇA

começa com um laço
uma condição lógica verdadeiro ou falso 
enquanto for verdade ele executa e volta para o teste fazendo um luping (laço) repetindo o camando enquanto a condição for verdade se mudar ele vai para o final e termina ou prossegue o código.

teste lógico verificando a condição no início. com "while"

var c = 1
while (c <= 6) {
    console.log('Tudo Bem?')
    c++
}
/Enquanto c for menor ou igual a 6 escreva tudo bem 6 vezes



teste lógico verificando a condição no final. com "do while"

var c = 1
 do {
    console.log('Tudo Bem?')
    c++
} while (c <= 6)

faça escreva tudo bem enquanto c for menor ou igual a 6.
